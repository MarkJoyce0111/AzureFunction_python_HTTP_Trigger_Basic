# AzureFunction_python_HTTP_Trigger_Basic
VSCode Azure Function. HTTP Trigger in python.

Use Postman or similar to send [POST]
With a raw body like:
{
    "name":"Mark",
    "code":"123",
    "country":"Oz"
}
